## 📂 Projects

### 1. Linear Regression
**Project:** Used Car Price Prediction
**What I covered**
*  Exploratory Data Analysis (EDA)
*  Data Cleaning & Preprocessing
*  Feature Engineering
*  Linear Regression Model
*  Model Evaluation
*  Residual Analysis
*  Identifying and fixing Data Leakage

**Final Results**
* **R² Score:** `0.6404`
* **Mean Squared Error (MSE):** `0.3504`
**Dataset**
* Vehicle Dataset from CarDekho
* Source: [Kaggle Dataset](https://www.kaggle.com/datasets/nehalbirla/vehicle-dataset-from-cardekho?utm_source=chatgpt.com)
